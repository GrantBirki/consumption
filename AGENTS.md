# consumption agent notes

## Design direction

- This site is intentionally hostile, maximalist web art. Prefer sensory overload over clarity.
- The palette should stay aggressively vibrant and unstable. Cursor movement should scramble the visible colors of the whole page immediately.
- Color motion should settle only after pointer movement stops. The page should feel reactive, not randomly jittery.
- `consumption.png` is a core visual motif and should remain prominent in multiple floating or repeated treatments.
- The spaced text `C O N S U M P T I O N` should remain a major background element of the composition.
- Motion, overlap, and visual noise are encouraged, but the implementation must stay free of console errors and avoid browser-crashing performance problems.
- When adding interaction logic, prefer extracting pure functions that can be unit tested.
